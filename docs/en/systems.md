<p>
<i>Attractor Builder, apart from allowing users to create custom systems, 
also enables the visualization and experimentation with twenty predefined 
dynamical systems commonly referred to as <b>strange attractors</b>. 
Each system page includes: a short animation, the system of differential 
equations, default parameter values, simulation settings, 
a summary of the system’s origin and attractor properties, and a reference to the
original publication.</i><br>
</p>

<p>
Select a system from the gallery below to view its detailed description.<br>
</p>
